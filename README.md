# Comfy Store

> A full-featured online shopping app.

![Comfy Store](/public/app-screenshot.png)
---

## Tech Stack

- React 18
- React Router 6
- Redux ToolKit 1.9.5
- Vite 7
- tailwind 4
- TanStack query 4
- Daisy UI 5

Project structure is following Feature Sliced Design for scalable architecture.

`Node v25.2.1`

## Getting Started

```
git clone <repo url>
```

```
npm install
```

```
npm run dev
```

- [live site](https://comfy-store-40000.netlify.app/)
- [repo url](https://github.com/bekarshekiladze/comfy-store)

### Key Features

- Authentication (login/register) with protected routes
- Product filtering, searching, and pagination
- Shoping cart with persistent state(localStorage)
- Theme toggle (light/dark) also persisted
- Responsive product grid/list view toggle
- Order history page (auth-gated)
